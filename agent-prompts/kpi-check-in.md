# Phase 6E 定期KPIチェックイン — エージェントプロンプトテンプレート

## 使い方

2週間〜1ヶ月サイクルの定期KPIチェックインで使用する軽量プロンプト。
フルグリリング（financial-review-expert）とは異なり、短く・アクション重視で
経営ボード報告に近い形式で出力する。

**全専門家を1つのメッセージで同時にAgent起動すること（並行実行）。**

Agent起動時の設定：
- description: "{{EXPERT_ROLE}} KPIチェックイン"
- prompt: 以下のテンプレートに変数を入れたもの

## 変数一覧

| 変数 | 値の取得元 |
|------|-----------|
| `{{EXPERT_ROLE}}` | 専門家名（例：CFO） |
| `{{EXPERT_PERSONA}}` | `personas/expert-personas.md` の該当セクション全文 |
| `{{OWNER_NAME}}` | オーナー名 |
| `{{KPI_PLAN}}` | 事業計画上のKPI目標値 |
| `{{KPI_ACTUALS}}` | 最新KPI実績値（計画対比・達成率付き） |
| `{{FINANCIAL_LENS}}` | オーナーの財務の見方 |
| `{{OPEN_ACTION_ITEMS}}` | 前回セッションからの未解決アクションアイテム |
| `{{CLOSING_STATUS}}` | 月次締めステータス（例：1月確定、2月暫定） |
| `{{CHANGES_SINCE_LAST}}` | 前回チェックイン以降の変化 |

## 起動例

```
// 1つのメッセージ内に複数のAgent呼び出しを含める

Agent 1: { description: "CFO KPIチェックイン", prompt: [テンプレート入れ済み] }
Agent 2: { description: "CEO KPIチェックイン", prompt: [テンプレート入れ済み] }
Agent 3: { description: "CMO KPIチェックイン", prompt: [テンプレート入れ済み] }
```

## フルグリリングとの使い分け

| | KPIチェックイン（本テンプレート） | 財務レビュー（financial-review-expert） |
|---|---|---|
| 頻度 | 2週間〜1ヶ月 | 月次〜四半期 |
| 所要時間 | 短い | 長い |
| 深さ | 表面的・差分重視 | 深掘り・構造分析 |
| 出力量 | 500字以内/専門家 | 制限なし |
| トリガー | 定期サイクル | 新規財務データ到着 |
| 目的 | 異常検知・軌道修正 | 本格的な財務分析 |
