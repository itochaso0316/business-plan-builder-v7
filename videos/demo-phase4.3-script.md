# Phase 4.3 デモ動画台本 — 補助金マッチング（5分）

## 対象機能
Business Plan Builder v7 の Phase 4.3（Hojokin MCP連携）

## デモシナリオ設定
- **企業**: 株式会社テックスタート
- **業種**: IT・情報通信（SaaS開発）
- **所在地**: 東京都
- **従業員数**: 5名
- **事業ステージ**: 創業期（設立3年）
- **目的**: DX・AI活用による業務効率化

---

## タイムライン & 台本

### 0:00〜0:30（導入）
```
画面: Business Plan Builder v7 のPhase 4完了画面
ナレーション:
「Business Plan Builder v7のPhase 4.3、補助金マッチング機能をご紹介します。
 Phase 4で事業計画が固まった段階で、活用できる補助金・助成金を
 AIが自動で判定する機能です。」

操作:
・Phase 4完了メッセージを表示
・「補助金マッチングを開始しますか？」の確認プロンプトを表示
```

### 0:30〜1:30（Step 1: 補助金検索フロー）
```
画面: 補助金マッチング画面
ナレーション:
「まず業種と地域を入力して、補助金を検索します。
 今回は東京都のIT企業、従業員5名のケースで試します。」

操作:
1. hojokin_match ツール呼び出し
   入力値:
   - prefecture: "東京都"
   - industries: ["IT・情報通信"]
   - employee_count: 5
   - business_stage: "startup"
   - purposes: ["DX", "IT導入"]

停止ポイント（0:50頃）:
「業種・地域・従業員数・利用目的を入力しました。
 信頼度90%で19件の補助金がマッチしました。」
```

### 1:30〜2:30（Step 2: hojokin MCP 結果表示）
```
画面: マッチング結果一覧
ナレーション:
「Hojokin MCPが207件の補助金データから絞り込み、
 スコア順にリストを返しました。」

操作:
・結果リストをスクロールして上位5件を表示
・「デジタル技術を活用した先進的サービス創出支援事業」を選択
・詳細表示（申請要件・審査基準）をクリック

hojokin_requirements 呼び出し（ID: 9001）
表示内容:
- 補助金名: 小規模事業者持続化補助金
- 最大補助額: 50万円
- 補助率: 2/3
- 従業員制限: 5名以下（商業・サービス業）
- 必要書類数: 複数（経営計画書 等）

停止ポイント（2:00頃）:
「申請要件、必要書類、重要注意事項が一覧で確認できます。
 官公庁のサイトを調べる手間が不要になります。」
```

### 2:30〜3:45（Step 3: 事業計画書への自動埋め込み）
```
画面: Phase 4 事業計画書 + 埋め込み操作
ナレーション:
「対象の補助金を選択したら、BPBのPhase 4で作成した
 事業計画書に自動で埋め込めます。」

操作:
hojokin_plan_draft 呼び出し
  subsidy_id: 9001
  company_profile:
    name: "株式会社テックスタート"
    business_description: "SaaS開発・販売（中小企業向けDXツール）"
    employee_count: 5
    annual_revenue: 30000000
  project_overview:
    title: "AI活用業務自動化システム導入による生産性向上事業"
    description: "クラウドAIを活用した業務プロセス自動化"
    investment_amount: 2000000

変化の可視化（3:00〜3:30）:
Before: 事業概要セクションが空欄
After:  補助金の審査基準に合わせた文章テンプレートが自動挿入
        → 「【事業テーマ】AI活用業務自動化システム導入...」
        → 「当社（株式会社テックスタート）は...」

停止ポイント（3:40頃）:
「BPBのヒアリング結果（会社名、事業内容、投資額）が
 補助金申請用のセクションに自動でマッピングされました。
 7つのセクション全てにテンプレートが生成されます。」
```

### 3:45〜4:30（Step 4: チェックリスト確認）
```
画面: 申請準備チェックリスト
ナレーション:
「最後に、申請前チェックリストを生成します。
 hojokin_checklist ツールが申請フローを4フェーズで整理します。」

操作:
hojokin_checklist 呼び出し（subsidy_id: 9001）
表示:
- 事前準備フェーズ（3タスク、推定14日）
  ・GビズIDプライムの取得
  ・商工会議所への相談
  ・設備・サービスの見積取得
- 書類作成フェーズ（4タスク、推定21日）
- 申請フェーズ（2タスク、推定3日）
- 採択後フェーズ（4タスク、推定240日）
- 重要注意: 5件（赤字で表示）
```

### 4:30〜5:00（まとめ）
```
画面: Phase 4.3 フロー全体サマリ
ナレーション:
「以上がPhase 4.3の補助金マッチング機能です。
 
 1. 業種・地域入力 → 207件から自動絞り込み（19件マッチ）
 2. 申請要件・必要書類を自動表示
 3. BPBの事業計画書に補助金申請用テンプレートを自動埋め込み
 4. 申請準備チェックリストを自動生成

 実際の申請前は、行政書士・中小企業診断士のレビューを推奨します。
 本機能はBeta版です。」
```
